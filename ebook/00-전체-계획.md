# React 19 실전 완벽 가이드 - 전자책 계획서

> **버전**: React 19.3.0 기반
> **최종 업데이트**: 2025-11-11
> **목표**: 시나리오 기반 학습으로 협업 작업 관리 플랫폼 "TaskFlow" 완성

---

## 📖 전자책 개요

### 목표
- React 19의 최신 기능과 모범 사례를 실전 중심으로 학습
- 하나의 완성된 애플리케이션을 단계적으로 구축
- 이전 버전과의 차이점 및 마이그레이션 가이드 제공

### 대상 독자
- React 기초 지식이 있는 개발자
- React 18에서 19로 업그레이드를 고려하는 개발자
- 최신 React 패턴과 모범 사례를 학습하고자 하는 개발자

### 프로젝트: TaskFlow
**협업 작업 관리 플랫폼**

#### 주요 기능
1. **사용자 인증 및 프로필 관리**
2. **프로젝트 생성 및 관리**
3. **작업(Task) CRUD**
4. **실시간 협업 및 낙관적 업데이트**
5. **댓글 및 첨부파일**
6. **대시보드 및 통계**
7. **검색 및 필터링**
8. **다크모드 및 접근성**

## 🎨 코드 스타일 가이드

### TypeScript
- 모든 코드는 TypeScript로 작성
- 명확한 타입 정의
- Interface vs Type 일관성

### 네이밍
- 컴포넌트: PascalCase (예: `TaskCard`)
- Hooks: camelCase with `use` prefix (예: `useTaskList`)
- 파일: kebab-case (예: `task-card.tsx`)

### 주석
- JSDoc 스타일
- 복잡한 로직에는 상세한 주석
- React 19 특화 기능은 `// React 19:` 주석 추가
